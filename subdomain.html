Finding Subdomain

subfinder -d target.com -o subdomains.txt

amass enum -passive -d target.com -o subdomains.txt

sublist3r -d target.com -o subdomains.txt

assetfinder --subs-only target.com > subdomains.txt

amass enum -active -d target.com -o subdomains.txt

findomain -t example.com

findomain -t example.com

dnsrecon -d example.com

Perform zone transfer:

dnsrecon -d example.com -t axfr

shodan search hostname:*.example.com

Search for subdomains via Certificate Transparency logs.

certspotter example.com

Combine multiple tools for better results:

subfinder -d example.com | assetfinder --subs-only | amass enum -d example.com | sort -u > final_subdomains.txt
